# Developer Portfolio

정적 HTML/CSS로 구성된 개인 개발 제품 포트폴리오입니다.

## 로컬 미리보기

macOS 기본 Python 간이 서버 예시:

```bash
cd /Users/telo.lee/dev/developer-page
python3 -m http.server 5173
```

브라우저에서 `http://localhost:5173` 접속.

## 배포 (GitHub Pages)

1. 이 디렉토리를 별도 리포지토리 루트로 사용하거나, 루트에 `index.html`이 있도록 push 합니다.
2. GitHub → Settings → Pages → Deploy from a branch → `main` / `/ (root)` 선택 후 저장.
3. 수 분 뒤 공개 URL이 생성됩니다.

## 포함 콘텐츠

- 메인: `index.html`, 스타일: `styles.css`
- 쉐도잉 마스터 가이드/개인정보: `guide-shadowing-master.html` → `guide-shadowing-master.md` 렌더링
- 앱 스토어 링크: 쉐도잉 마스터 (`https://apps.apple.com/kr/app/%EC%89%90%EB%8F%84%EC%9E%89%EB%A7%88%EC%8A%A4%ED%84%B0/id6751806807`)
